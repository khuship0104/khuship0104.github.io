LLM Agents

Bidding War Simulation

Independent Project · Second-price auction · Dec 2025

A multi-agent second-price (Vickrey) auction simulator. Strategic bidder agents combine a best-response approximator with an LLM call — grounding each bid in an actual expected-utility calculation rather than letting the model guess — while heuristic agents bid a simple value fraction. Outputs are validated with Pydantic and aggregated across simulation runs.

20-round sim → mean clearing price 0.50 · win share B1 10% / B2 55% / B3 35%
